# 龍の選定基準

> 対応ファイル: `hidden/selection-criteria.html`
> 発見キーワード: 「選定」（Lv2→3）
> 役割: 7つの選定基準がプレイヤー自身に当てはめられ、全項目PASSとして開示される。エンディングへの直接的な導線。

---

## 選定基準項目（7項目）

| 番号 | 基準 | 内容 | 判定 |
|---|---|---|---|
| 01 | CURIOSITY | 隠されたものに対し、自発的に調査・追跡する能力を持つこと | ✓ PASS — このページを発見した時点で充足 |
| 02 | PERSISTENCE | 困難なルートを経由しても調査を諦めないこと | ✓ PASS — 複数のキーワードを辿った記録あり |
| 03 | INSIGHT | 表面的な情報の裏にある構造を読み取れること | ✓ PASS — 縦読み・隠しマークを解読 |
| 04 | LOYALTY | 龍牌会の理念に反しない価値観を持つこと | ✓ PASS — 龍牌会の理念は「来た者が龍である」のみ。調査行為はこれと矛盾しない |
| 05 | ADAPTABILITY | 未知の状況・情報に対し柔軟に対処できること | ✓ PASS — このサイトへの適応速度より判定 |
| 06 | ORIGIN | 「外から来る者」であること（先代総龍の預言より） | ✓ PASS — サイト来訪者は定義上「外から来る者」 |
| 07 | RECOGNITION | ゆめが認めた者であること | ✓ PASS — RENPAI バックドアを通過した記録あり |

## テコ入れメモ

- 7つの基準がすべて「PASS」として開示される演出は効果的（順番に表示されるアニメーションあり）
- 弱点：「LOYALTY（龍牌会の理念に反しない価値観）」がPASSになるのはおかしくないか？ プレイヤーは龍牌会に批判的に調査しているはず。この基準の説明が都合よすぎる
- 「ゆめが認めた者であること」（基準07）がRENPAI通過と結びついている設計は美しい
- 問いを追加できる余地：「基準を全部満たした。でも、あなたは龍になりたいか？」という問いがない。受け身の承認だけで終わっている
