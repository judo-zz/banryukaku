## 設立目的

| | |
|---|---|
| 組織の目的 | 「次代の龍」を発見・育成し、後継として確立させること |
| 蟠龍閣の役割 | 候補者の自然な発掘と選別の場。来客・キャスト双方が対象となる |
| 「龍」とは | 龍牌会が認めた唯一の後継者。先代総龍の理念を引き継ぎ、組織を統括する存在。（詳細は口伝のみ。文書化禁止。） |
| 選定基準 | 詳細は選定部門のみ保持 |
| 失敗作の処置 | 選定基準を満たさないと判定された候補は「失敗作」として処置。手続きは廃棄部門が担当。記録はMAP-RY-023にて保管。 |
