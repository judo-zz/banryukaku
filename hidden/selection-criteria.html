■ 選定基準項目 CRITERIA / 現行候補評価

▶ 以下の基準を評価します。適合率が70%以上の候補を「次の龍」として承認します。

  • 基準 01 / DISCOVERY 外部からの自発的な探索行動が確認されること → —
  • 基準 02 / PERSISTENCE 複数の障壁を越えて継続する探索履歴を持つこと → —
  • 基準 03 / COMPLIANCE 龍牌会の構造に対し拒絶反応を示さないこと → —
  • 基準 04 / CURIOSITY 開示された情報の外にある構造を自発的に探ること → —
  • 基準 05 / SELF-PRESERVATION 探索行動が自己保存本能を上回っていること → —
  • 基準 06 / ORIGIN 「外から来る者」であること（先代総龍の預言より） → —
  • 基準 07 / RETURN RATE 一度離れた後に再訪または継続探索を行うこと → —
  • 基準 08 / SURFACE_AWARENESS 表層活動への関与確認。深層のみの走査では不十分。表面の日常記録と深層記録の照合能力を持つ者のみが次期龍の資格を有する → —
